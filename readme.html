Guide

BioLog Mobile

スマホで使う、オフライン対応の健康記録PWAです。

Disclaimer

免責事項

BioLog Mobile は日々の健康記録を補助するための個人向けツールです。医療行為、診断、治療、助言を目的としたものではありません。

体調不良、異常値、健康上の不安がある場合は、医師などの専門家に相談してください。

PWA

アプリのように使えます

公開URLをブラウザで開き、ホーム画面に追加して使います。App Store や Google Play からインストールするアプリではありません。

Storage

端末内に保存

記録は端末のブラウザ内 IndexedDB に保存されます。外部サーバーには送信されません。

Features

主な機能

  • 今日の健康記録入力
  • 日付を指定した新規登録、編集
  • 履歴の確認、編集、削除
  • 体重、体温、血圧、脈拍、体脂肪率、基礎代謝量、筋肉量のグラフ表示
  • JSONバックアップ書き出し、読み込み
  • UTF-8 CSV読み込み
  • ライトモード / ダークモード

How To Use

基本操作

今日: 今日の測定値や食事ログ、行動ログ、メモを保存します。

日付編集: 対象日を選んで読み込み、過去日や任意の日付を登録・編集します。

履歴: 保存済み記録の確認、編集、削除を行います。

グラフ: 記録済みデータの推移を確認します。

バックアップ: JSONの書き出し、JSON/CSVの読み込み、全データ削除を行います。

Backup

バックアップは大切です

ブラウザのサイトデータを削除すると、保存した記録も削除されます。機種変更、ブラウザ変更、端末初期化の前には、必ずJSONバックアップを書き出してください。

iPhoneでは、Safariで見えるデータとホーム画面版で見えるデータが分かれて見える場合があります。ホーム画面アイコンを削除・再追加する前には、必ずJSONバックアップを作成してください。

バックアップJSONや個人データを含むCSVは、GitHubなどの公開場所へアップロードしないでください。

CSV

CSV読み込み

UTF-8 CSVを読み込めます。既存BioLogの一覧タブから書き出したCSVにも対応しています。

CSVは外部データ取り込み用です。完全な復元用途にはJSONバックアップを使ってください。
